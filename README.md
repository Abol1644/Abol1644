## Engineering Principles

| Principle                    | What it means                                                  |
| ---------------------------- | -------------------------------------------------------------- |

| Performance is a feature     | Fast interfaces feel more trustworthy.                         |

| Simple systems survive       | Clean systems age better than clever ones.                     |
